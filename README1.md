# Task 1 - Data Structures Implementation

## Objective
The objective of this project is to implement fundamental data structures in C++ and demonstrate their basic operations using menu-driven programs.

## Data Structures Implemented
1. Stack
2. Queue

## Features

### Stack
- Push
- Pop
- Peek
- Display

### Queue
- Enqueue
- Dequeue
- Peek
- Display

## Language Used
- C++

## Project Files
```
Task1_DataStructures/
├── stack.cpp
├── queue.cpp
└── README.md
```

## How to Compile and Run

### Stack

Compile:
```bash
g++ stack.cpp -o stack
```

Run:
```bash
stack.exe
```

### Queue

Compile:
```bash
g++ queue.cpp -o queue
```

Run:
```bash
queue.exe
```

## Time Complexity

### Stack
| Operation | Time Complexity |
|-----------|-----------------|
| Push | O(1) |
| Pop | O(1) |
| Peek | O(1) |
| Display | O(n) |

### Queue
| Operation | Time Complexity |
|-----------|-----------------|
| Enqueue | O(1) |
| Dequeue | O(1) |
| Peek | O(1) |
| Display | O(n) |

## Sample Operations

### Stack
- Push an element
- Pop an element
- View the top element
- Display all elements

### Queue
- Enqueue an element
- Dequeue an element
- View the front element
- Display all elements

## Conclusion
This project demonstrates the implementation of two fundamental linear data structures, Stack and Queue, using C++. The programs are menu-driven and perform insertion, deletion, and display operations efficiently.
